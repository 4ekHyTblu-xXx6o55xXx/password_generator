# 🔐 Password Generator

[![Python 3.6+](https://img.shields.io/badge/python-3.6+-blue.svg)](https://www.python.org/downloads/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg)](http://makeapullrequest.com)

**Криптографически безопасный генератор паролей** с поддержкой:
- обычных паролей любой длины и сложности;
- паролей-фраз из случайных слов (стиль XKCD);
- исключения неоднозначных символов (`l`, `I`, `1`, `0`, `O` и др.);
- расчёта энтропии и оценки стойкости;
- цветного вывода в терминале.

---

## 🎮 Использование
### Базовый пароль (длина 16, все типы символов)
```bash
pwdgen
```
